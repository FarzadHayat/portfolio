# Pet Ghost Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a friendly Casper-style ghost (`👻`) to the bottom-left corner of the portfolio that pops up witty section-specific speech bubbles as the visitor scrolls, and bounces when clicked.

**Architecture:** A single self-contained `PetGhost.astro` component with inline CSS and a `<script>` tag. No new dependencies. Integrated into `Layout.astro` alongside the existing `FabFlower` and `Dock` components. Desktop-only.

**Tech Stack:** Astro, vanilla JS, CSS keyframes, IntersectionObserver API

- [ ] **Step 1: Create `src/components/PetGhost.astro` with this exact content**

```astro
---
// PetGhost — friendly Casper-style pet for the portfolio
// Desktop only (hidden lg:block). No props, no dependencies.
---

<div
  id="pet-ghost"
  class="fixed bottom-6 left-6 z-50 hidden lg:block"
>
  <div
    id="ghost-bubble"
    class="ghost-bubble pointer-events-none"
    aria-live="polite"
  >
    Welcome, mortal 👻
  </div>
  <button id="ghost-emoji" type="button" class="ghost-emoji cursor-pointer select-none" aria-label="Pet ghost">
    👻
  </button>
</div>

<style>
  /* ── Layout ─────────────────────────────────────────── */
  .ghost-emoji {
    font-size: 2.5rem;
    line-height: 1;
    display: block;
    animation: ghost-float 3s ease-in-out infinite;
    transition: filter 0.2s ease;
  }

  .ghost-emoji:hover {
    filter: drop-shadow(0 0 8px rgba(167, 139, 250, 0.6));
  }

  /* ── Speech bubble ──────────────────────────────────── */
  .ghost-bubble {
    background: white;
    color: #1e1e2e;
    font-size: 0.8rem;
    font-weight: 600;
    padding: 6px 12px;
    border-radius: 999px;
    white-space: nowrap;
    box-shadow: 0 4px 20px rgba(0, 0, 0, 0.35);
    /* viewport-anchored — immune to container-width edge cases */
    position: fixed;
    bottom: 5.5rem; /* ghost-bottom(1.5rem) + emoji-height(2.5rem) + gap(1.5rem) */
    left: 2rem;
    transform: translateY(-4px);
    opacity: 0;
    transition:
      opacity 0.25s ease,
      transform 0.25s ease;
  }

  /* downward tail — points at ghost emoji center */
  .ghost-bubble::after {
    content: '';
    position: absolute;
    bottom: -5px;
    left: 1.25rem;
    transform: translateX(-50%);
    border-left: 6px solid transparent;
    border-right: 6px solid transparent;
    border-top: 6px solid white;
  }

  .ghost-bubble.visible {
    opacity: 1;
    transform: translateY(0);
  }

  /* ── Animations ─────────────────────────────────────── */
  @keyframes ghost-float {
    0%, 100% { transform: translateY(0px) rotate(-3deg); }
    50%       { transform: translateY(-10px) rotate(3deg); }
  }

  @keyframes ghost-bounce {
    0%   { transform: scale(1) rotate(0deg); }
    30%  { transform: scale(1.4) rotate(-10deg); }
    60%  { transform: scale(0.9) rotate(5deg); }
    100% { transform: scale(1) rotate(0deg); }
  }

  .ghost-emoji.bouncing {
    animation: ghost-bounce 0.4s ease forwards;
  }
</style>

<script>
  // ── Section quips ──────────────────────────────────────
  // Inactive sections are included for when they get re-enabled.
  const SECTION_QUIPS: Record<string, string> = {
    hero:           'Welcome, mortal 👻',
    projects:       'These projects are to die for 💀',
    work:           'Impressive résumé for a living person',
    education:      'Still smarter than me 🎓',
    certifications: 'Certified haunted engineer 🏅',
    hackathons:     'Hacking the afterlife since forever',
    spotify:        'This human has great taste in music 🎵',
    about:          'A human with good taste!',
    blog:           'Wise words from beyond the grave 📖',
    contact:        'Go on... say hello 👋',
  };

  const CLICK_MESSAGES = [
    'Boo! 👀',
    'You found me!',
    'Pet accepted ✅',
    'Hehe 👻',
    "Stop that... actually don't",
  ];

  // ── DOM refs ───────────────────────────────────────────
  const ghost   = document.getElementById('pet-ghost')!;
  const bubble  = document.getElementById('ghost-bubble')!;
  const emoji   = document.getElementById('ghost-emoji')!;

  let bubbleTimer: ReturnType<typeof setTimeout> | null = null;
  let clickCooldown = false;

  // ── Bubble helpers ─────────────────────────────────────
  function showBubble(text: string, durationMs = 4000) {
    if (bubbleTimer) clearTimeout(bubbleTimer);
    bubble.textContent = text;
    bubble.classList.add('visible');
    bubbleTimer = setTimeout(() => bubble.classList.remove('visible'), durationMs);
  }

  // ── Section observer ───────────────────────────────────
  function observeSections() {
    // hero has no id — target by class
    const heroEl = document.querySelector('section.hero');
    const namedEls = Object.keys(SECTION_QUIPS)
      .filter((k) => k !== 'hero')
      .map((id) => document.getElementById(id))
      .filter(Boolean) as HTMLElement[];

    const targets: Array<{ el: Element; quip: string }> = [];
    if (heroEl) targets.push({ el: heroEl, quip: SECTION_QUIPS.hero });
    namedEls.forEach((el) => targets.push({ el, quip: SECTION_QUIPS[el.id] }));

    const observer = new IntersectionObserver(
      (entries, obs) => {
        entries.forEach((entry) => {
          if (!entry.isIntersecting) return;
          const target = targets.find((t) => t.el === entry.target);
          if (target) showBubble(target.quip);
          obs.unobserve(entry.target); // fire once per section per page load
        });
      },
      { threshold: 0.3 }
    );

    targets.forEach(({ el }) => observer.observe(el));
  }

  // ── Click handler ──────────────────────────────────────
  emoji.addEventListener('click', () => {
    if (clickCooldown) return;
    clickCooldown = true;

    const msg = CLICK_MESSAGES[Math.floor(Math.random() * CLICK_MESSAGES.length)];
    showBubble(msg, 2000);

    emoji.classList.remove('bouncing');
    // Force reflow so re-adding the class restarts the animation
    void (emoji as HTMLElement).offsetWidth;
    emoji.classList.add('bouncing');

    setTimeout(() => {
      emoji.classList.remove('bouncing');
      clickCooldown = false;
    }, 2000);
  });

  // ── Init ───────────────────────────────────────────────
  // Use 'astro:page-load' to support Astro's View Transitions
  document.addEventListener('astro:page-load', observeSections);
</script>
```

- [ ] **Step 2: If a section bubble does not appear**

Open browser DevTools console. Look for errors. Common issues:
- Section has no `id` matching the key in `SECTION_QUIPS` → check the rendered HTML for that section's actual id
- `IntersectionObserver` threshold too high for short sections → lower threshold from `0.3` to `0.15` in `PetGhost.astro`
